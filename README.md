# Todo API with Laravel
Laravel app to make a todo api. 

<!-- step -->
### Database
App is connected to **MySQL** database using the .env file. 

<!-- step -->
#### Routes
Following routes to CRUD operations
* [GET] /api/todos
* [POST] /api/todos
* [GET] /api/todos/{id}
* [PUT] /api/todos/{id}
* [DELETE] /api/todos/{id}
